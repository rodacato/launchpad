Read `skills/sdd-tasks/SKILL.md` and follow its instructions exactly.

The change name is provided in `$ARGUMENTS`. The skill requires the plan
to be `accepted` — it reads plan.md to derive the task list.

```
Usage: /lifecycle:sdd-tasks <change-name>

Examples:
  /lifecycle:sdd-tasks jwt-exp-validation
```
