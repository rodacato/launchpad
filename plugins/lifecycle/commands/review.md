---
description: "Run a multi-axis code review on a GitHub PR using gh CLI"
---

Read `skills/code-review/SKILL.md` and follow its instructions exactly.

The PR number or URL is provided in `$ARGUMENTS`. If `$ARGUMENTS` is empty, respond with:

```
Usage: /lifecycle:review <PR-number-or-URL>

Example:
  /lifecycle:review 42
  /lifecycle:review https://github.com/owner/repo/pull/42
```
