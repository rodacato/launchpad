---
description: "Create or update the user-level Claude Code output style (voice)"
---

Read `skills/voice/SKILL.md` and follow its instructions exactly.

An optional style name can be provided in `$ARGUMENTS`. If present, use it as
the name for the output style file. If empty, the skill will ask the user for
a name during Step 2.

```
Usage: /philosophy:voice [style-name]

Examples:
  /philosophy:voice
  /philosophy:voice adrian
  /philosophy:voice rodacato
```
