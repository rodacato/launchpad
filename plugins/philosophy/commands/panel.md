---
description: "Consult the experts advisory panel on a decision or design question"
---

Read `skills/experts/SKILL.md` and follow its instructions exactly.

The question, decision, or topic for the panel is provided in `$ARGUMENTS`. If
`$ARGUMENTS` is empty, respond with:

```
Usage: /philosophy:panel <question or topic>

Examples:
  /philosophy:panel Should this module structure be flat or nested?
  /philosophy:panel Architect + DevX: review the new command dispatch pattern
  /philosophy:panel threat model the local plugin install path

You can also target specific experts by name (Marco, Sofía, Kai, Léa, Arjun, Elena)
or consult the full panel.
```
